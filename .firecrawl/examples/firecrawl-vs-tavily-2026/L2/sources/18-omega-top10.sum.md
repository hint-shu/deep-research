## Key facts

- **14+ providers now offer programmatic AI search APIs** (market fragmentation)
- "Radically different pricing models, latency profiles, architectural assumptions about how agents should consume web data"
- **Category split:**
  - **Category A (ready answer):** Perplexity (executes search, reads results, synthesizes answer with own LLM, returns with citations), OpenAI, Anthropic, Google, Brave Summarizer
  - **Category B (raw materials):** **Tavily, Exa, You.com, Serper, Firecrawl** — structured results (some full page content), agent synthesizes with own LLM
- **Key decision variable:** "Does your agent already have a synthesis LLM?"
  - If yes (Claude/GPT/Gemini) → Category B достаточно
  - If no → Category A дешевле/быстрее (all-in-one)

## Key quotes
