## Key facts

- **Feb 2026:** Nebius (Amsterdam AI cloud infra) acquired Tavily
- **Tavily p95 latency:** 3.8-4.5s (vs Exa 1.4-1.7s) — **Tavily медленнее**
- **Tavily fastest search latency:** 210-420ms
- **Tavily rate limits:** 1,000 req/min
- Tavily: **не имеет собственного index**, использует внешние (Exa имеет свой index из billions of documents)
- Tavily bundled pricing — "free raw content included" — может быть дешевле при full-page text use case
- SDK: Python, JavaScript
- **MCP support:** yes
- Framework integrations: LangChain (default), LlamaIndex, CrewAI, Vercel AI SDK, Dify, Make, n8n, Zapier, 20+
- **No OpenAI SDK compatibility** (в отличие от Exa)
- Free tier: 1,000 credits/month
