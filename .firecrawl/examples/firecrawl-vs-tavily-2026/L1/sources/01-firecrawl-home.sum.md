## Key facts

- Позиционирование: "API to search, scrape, and interact with the web at scale. Also open source."
- **Шесть endpoints**: Search, Scrape, Map, Crawl, Agent, Interact
- Используется: Shopify, Canva, Apple, Zapier, Replit, DoorDash, Alibaba, You.com, Lovable, Gamma
- Заявлено **80,000+ компаний** и **500,000+ разработчиков**
- Open-source с AGPL-3.0 лицензией, **44,700+ GitHub stars**
- Цель — "clean web data" для AI-агентов в LLM-ready формате (Markdown, JSON)
- `/agent` endpoint — автономная навигация и research без написания селекторов
- `/interact` endpoint — скрейп страницы + действия (клики, формы, извлечение за динамическим контентом)
- Есть CLI (`firecrawl scrape`, `firecrawl search`, `firecrawl crawl`)
- Python/Node.js/Rust/Go SDK

## Key quotes
